## 一、选美国云主机，先想清楚你到底要什么

很多人挑美国云主机的第一步就错了——上来就比价格。其实价格是最后才该看的事。先把下面三个问题问自己一遍：

**你的访问用户主要在哪？** 如果是面向海外、做外贸、跑全球业务，那随便一条 BGP 大带宽线路都能用，拼的是机房稳定性和带宽大小；如果用户主要在国内、自己也要从国内 SSH 上去折腾，那就必须看"回程优化线路"——也就是后面要讲的 CN2 GIA、9929、CMIN2 这些。

**你要跑什么负载？** 挂个小博客、跑个轻量面板、做代理中转，1 核 1G 就够；要跑数据库、爬虫、电商站、并发稍微高一点的应用，2 核 4G 起步更稳；真要扛业务、跑容器集群，4 核 8G 是分水岭。

**你对延迟敏感还是对带宽敏感？** 优化线路普遍延迟低、丢包少，但带宽通常被限制在 200M–500M；普通 BGP 大带宽能给你 1Gbps 甚至更多，但晚高峰可能丢包。鱼和熊掌，自己选。

把这三件事想明白，再看下面这张"线路翻译表"，整个人会清爽很多。
